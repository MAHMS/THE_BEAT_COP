The contents of these files are loaded in a specific order. If you edit these text files, don't rearrange the lines!
This was the easiest way to save / load data into PD without using an external.
This applies to all text files, including the midi-map, settings, patterns, and samples

NOTE:
If you want to add more instances of The Beat Cop, 
simply copy paste one of the folders, 
then rename it to a new number such as tbc-6 , tbc-7 , tbc-8 , ... tbc-9998 etc.

tbc-9999 is used for the help file.

You could probably go as high as the maximum value of an int on your system.
